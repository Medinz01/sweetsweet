[Body Text] 
[Body Text] 
[Body Text] 
[Body Text] ARRANGEMENT OF CONTENTS:
[Body Text] 
[Body Text] The sequence of the project report should be arranged and bound should be as follows:
[Body Text] 
[Body Text] Cover Page & Title Page
[Body Text] Certificate
[Body Text] Acknowledgement
[Body Text] Abstract
[Body Text] List of Tables
[Body Text] List of Figures
[Body Text] List of Acronyms (if any)
[Body Text] Table of Contents
[Body Text] Chapters (1 to 7)
[Body Text] Appendices (Sample Code – 3 to 5 pages only)
[Body Text] References
[Body Text] 
[Body Text] Important Note:
[Body Text] The Project Report should have minimum 50 pages (Chapter 1 to Chapter 7) excluding the Reference Section.
[Body Text] 
[Body Text] Kindly instructed to follow the enclosed Guidelines regarding the Documentation
[Body Text] 
[Body Text] Page size – A4
[Body Text] Margin – 3.5 cm Left, and 2.5cm on Right, Top and Bottom
[Body Text] Font Type – Time New Roman
[Body Text] Font Size: running text – 12, Main Headings -14 and Bold, Sub-Headings -12 and Bold
[Body Text] Line Spacing – 1.5
[Body Text] No (extra) spacing on First and Last Line of the paragraphs.
[Body Text] Minimum 8 to 10 lines of content (in the Word Document) can be considered as a Paragraph.
[Body Text] Figure Title (Caption) should be placed below the Figure and Table Title (Caption) should be placed above the Table.
[Body Text] Sequential Numbering (for example: Figure 1.1, Figure 2.1, Figure 3.1 and so on) of Figures should be followed.
[Body Text] Sequential Numbering (for example: Table 1.1, Table 2.1, Table 3.1 and so on) of Tables should be followed.
[Body Text] While incorporating Figures, Graphs and Tables, description of concerned Figures and Tables should be incorporated (added) in the Documentation.
[Body Text] During the Citation, References are represented only by Square Brackets, for Example [2]
[Body Text]
